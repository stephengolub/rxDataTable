rxItemsPerPage
directive in module rxDataTable

Description

Directive that takes in a page-tracking object and a label for what to call items and outputs a select box that allows you to change how many items in the list to show at a time

Usage

as element:
<rx-items-per-page
       pager-object="{Object}"
       label="{string='Items'}">
</rx-items-per-page>

Parameters

ParamTypeDetails
pager-objectObject

This is the page tracking service instance to be used for this directive

labelstring='Items'

This is the name of the items that you are restricting. It defaults to 'Items' and thus outputs 'Items per page'
