PageTracking
service in module rxDataTable

Description

This is the data service that can be used in conjunction with the pagination objects to store/control page display of data tables and other items.

Properties

  • MAX_PER_PAGE

    This is a value that is used in the iteration function to generate the item size list.

  • MIN_PER_PAGE

    This is a value that is used in the iteration function to generate the item size list.

  • ITEMS_PER_PAGE_STEP

    This is a value that is used in the iteration function to generate the item size list.

  • itemsPerPage

    This is the current setting for the number of items to display per page

  • pagesToShow

    This is the number of pages to show in the pagination controls

  • itemSizeList

    This is where the rxItemsPerPage Directive will store it's list of items per page

  • pageNumber

    This is where the current page number is stored.

  • pageInit

    This is used to determine if the page has been initialzed before.

  • total

    This is the total number of items that are in the data set

  • showAll

    This is used to determine whether or not to use the pagination or not.
