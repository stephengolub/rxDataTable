Page
filter in module rxDataTable

Description

This is the pagination filter that is used to limit the number of pages shown

Usage

In HTML Template Binding

{{ Page_expression | Page }}

In JavaScript

$filter('Page')(pager-object)

Parameters

ParamTypeDetails
pager-objectObject

The instance of the PageTracking service. If not specified, a new one will be created.

Returns

Array

The list of page numbers that will be displayed.
